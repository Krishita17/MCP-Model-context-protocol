# Security Policy

## Responsible Use

This project contains **offensive security tools** (MCPoisoner) intended exclusively for:

- Authorized penetration testing
- Academic security research
- Defensive security evaluation
- Coordinated Vulnerability Disclosure (CVD)

**Do NOT use these tools against systems you do not own or have explicit written authorization to test.**

## Reporting Vulnerabilities

If you discover a security vulnerability in this project:

1. **Do NOT** open a public GitHub issue
2. Email: krishita1710@gmail.com
3. Include: description, reproduction steps, impact assessment
4. Expected response time: 48 hours

## Coordinated Vulnerability Disclosure

All zero-days discovered during this research follow a 90-day CVD protocol:

1. Private notification to the affected vendor
2. Coordinated patch timeline
3. Public disclosure with CVE assignment after patch availability
